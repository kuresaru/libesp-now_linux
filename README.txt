A library to communicate to esp32/esp8266 esp-now protocol with normal wlan adapter in linux.
Tested with esp8266 ESP8266_RTOS_SDK, Debian GNU/Linux 12 kernel 6.1.0-11-amd64.

see https://github.com/kuresaru/ks-espnow-gateway for example.


packet filter reference: https://github.com/thomasfla/Linux-ESPNOW
